How to Embed a Video in Your Website

A step-by-step guide to adding service-hosted videos to your web pages

Embedding videos from services like YouTube, Vimeo, or Dailymotion can enhance your website's content. This tutorial will walk you through the process using YouTube as an example.

Step 1: Find the Video You Want to Embed

Navigate to YouTube and find the video you'd like to embed on your website. Make sure the video has embedding enabled (most do by default).

Locate the video on YouTube

Step 2: Click the "Share" Button

Below the video player, click the "Share" button to open the sharing options.

Click the Share button under the video

Step 3: Select "Embed" Option

In the sharing options that appear, click on the "Embed" tab to access the embed code.

Select the Embed tab from sharing options

Step 4: Copy the Embed Code

YouTube will provide you with an iframe embed code. You can click "Copy" to copy it to your clipboard.

Copy the provided iframe code

Step 5: Paste the Code in Your HTML

In your website's HTML file, paste the copied code where you want the video to appear. The code will look something like this:

<iframe width="560" height="315" src="https://www.youtube.com/embed/VIDEO_ID" frameborder="0" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture" allowfullscreen></iframe>

Paste the code in your HTML file

Step 6: Save and View Your Page

Save your HTML file and open it in a browser to see the embedded video. You can adjust the width and height attributes in the iframe tag to change the player size.

The final embedded video on your webpage
